Gives you additional Analytics information about your web site visitors.

== Description ==
Just install, customize and enjoy!

== Installation ==
1. Upload \"WebThunder\" folder to the \"/wp-content/plugins/\" directory.
1. Activate the plugin through the \"Plugins\" menu in WordPress.
